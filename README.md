# KIRV Engineering Studio

Precision software systems for complex problems.

## About

KIRV is an engineering studio building foundational software infrastructure. This repository contains the source code for the KIRV corporate website.

## Design System

KIRV implements a custom design system with an architectural, industrial aesthetic:

- **Typography**: Space Grotesk (display) + JetBrains Mono (technical)
- **Colors**: Neutral architectural palette (Cream, Stone, Charcoal, Graphite)
- **Border Radius**: 0px (rectangular design language)

## Development

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```
